# 行内元素水平居中

# 1.text-align

<div class="parent">
  <span class="child">蒜头王八</span>
</div>

*{
  box-sizing: border-box;
  margin: 0;
  padding: 0;
}
.parent{
  background-color: skyblue;
  text-align: center;
}
.child{
  background-color: red;
}

# 2.fit-content

dom不变

为父元素宽度设置fit-content

.parent{
  background-color: skyblue;
  width: fit-content; /*适应内容的宽度*/
  margin: auto;
}

# 块级元素水平垂直居中

# 1.position定位

通过transform属性设置

.parent{
  background-color: skyblue;
  height: 200px;
  position: relative;
}
.child{
  background-color: red;
  width: 100px;
  height: 100px;
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%,-50%);
}

通过设置margin

.child{
  background-color: red;
  width: 100px;
  height: 100px;
  position: absolute;
  top: 50%;
  left: 50%;
  margin-top:-50px;
  margin-left:-50px;
}

把left、top、right、bottom设置为0，margin设置为auto

.child{
  background-color: red;
  width: 100px;
  height: 100px;
  position: absolute;
  left: 0;
  right: 0;
  top: 0;
  bottom: 0;
  margin: auto;
}

通过calc函数

.child{
  background-color: red;
  width: 100px;
  height: 100px;
  position: absolute;
  left: calc(50% - 50px);//'-'两边注意空格
  top: calc(50% - 50px);
}

# 2.设置padding

父元素设置

.parent{
  background-color: skyblue;
  padding: 10px;
}
.child{
  background-color: red;
  height: 100px;
}

子元素通过calc设置padding，注意box-sizing是content-box

*{
  box-sizing: content-box;//注意
  margin: 0;
  padding: 0;
}
.parent{
  background-color: skyblue;
  height: 200px;
  width: 200px;
  /* text-align: center; */
}
.child{
  background-color: red;
  height: 100px;
  width: 100px;
  padding: calc((100% - 100px) / 2);
  background-clip: content-box;
}

# 3.flex 布局

.parent{
  background-color: skyblue;
  height: 200px;
  display: flex;
  justify-content: center;
  align-items: center;
}
.child{
  background-color: red;
  height: 100px;
  width: 100px;
}

# 4.grid布局

.parent{
  background-color: skyblue;
  height: 200px;
  display: grid;
}
.child{
  background-color: red;
  height: 100px;
  width: 100px;
  align-self: center;
  justify-self: center;
}

# 5.设置伪元素

父元素设置text-align来水平居中

子元素和伪元素display要设置为inline-block

.parent{
  background-color: skyblue;
  height: 200px;
  text-align: center;
}
.child{
  background-color: red;
  height: 100px;
  width: 100px;
  display: inline-block;
  vertical-align: middle;
}
.parent::after{
  content: "";
  display: inline-block;
  height: 100%;
  vertical-align: middle;
}

# 6.借助table-cell

.parent{
  background-color: skyblue;
  height: 200px;
  width: 200px;
  display: table-cell;
  text-align: center;
  vertical-align: middle;
}
.child{
  background-color: red;
  height: 100px;
  width: 100px;
  display: inline-block;
}

# 7.transform + margin

.parent{

  width: 200px;
  overflow: hidden;
}
.child{
  background-color: red;
  height: 100px;
  width: 100px;
  margin: 50% auto;
  transform: translateY(-50%);
}
